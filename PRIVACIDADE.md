# Política de Privacidade – Bot Financeiro

Este aplicativo é de uso pessoal e privado, utilizado para organização financeira dos usuários autorizados.

## Dados coletados
- Número de telefone
- Mensagens enviadas pelo usuário
- Registros financeiros informados voluntariamente

## Uso dos dados
Os dados são utilizados exclusivamente para controle financeiro do usuário.
Não há compartilhamento com terceiros.
